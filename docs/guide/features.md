# 機能一覧

## VR レンダリング
両眼立体視 + HMD ヘッドトラッキング（OpenXR backend）。ゲームを一人称 VR として描画します。

## シームレスな遷移
シーン遷移・環境切替で視界を保ち、遷移絵柄も VR 空間に表示します。暗転で酔わないよう配慮しています。

## Comfort 設定
World Scale / 目の高さ / 酔いカメラ抑制 / フレームペーシング（リプロジェクション由来のちらつき対策）。
詳細は [設定](/guide/configuration) を参照。

## ワールド UI 投影
ゲーム UI を VR 空間の曲面パネルに投影し、コントローラのレーザーで操作します。
位置・距離・サイズ・曲面を調整でき、固定位置の保存も可能です（[操作方法](/guide/controls) 参照）。

## 手元モデル
コントローラ実機モデル / アニメ調ハンドモデルを表示・切替できます。

## ジェスチャ & ゲームパッド入力
ゲームパッド準拠のボタン割当に加え、カラオケの振り入力（左手＝タンバリン / 右手＝ガヤ）、
カルタのスティック操作など、ミニゲーム別のジェスチャに対応します（[操作方法](/guide/controls) 参照）。

## 空間音声
Steam Audio (HRTF) によるキャストボイスの 3D 化。声の方向感が得られます。

## 視線追従
顔 / 虹彩の HMD 追従（Head / Eye Look-At）。キャストがプレイヤーの方を見ます。

## ミニゲーム対応
チェキ・手押し相撲・あ〜ん・カラオケ・カルタ・ドリンク等の VR 操作に対応します。
各操作は [操作方法](/guide/controls) のミニゲーム別ジェスチャを参照してください。
