# 設定

BG2VR の設定は 2 通りの方法で変更できます。

## ゲーム内 F10 パネル

ゲーム中に **F10** キー（または両スティック押し込み長押し）で設定パネルを開きます。
スライダー・トグルをその場で変更でき、多くの項目は**即座に反映**されます（再起動不要）。

VR 内では、設定パネルにレーザーを当ててトリガーでクリックするか、右スティックで項目移動・右 A ボタンで決定します。

## 設定ファイル（.cfg）

ゲームを一度起動すると `BepInEx\config\BG2VR.cfg` が生成されます。テキストエディタで直接編集できます。
変更はゲーム再起動後に反映されます。

## 全設定の一覧

各設定の名前・既定値・説明は [設定リファレンス](/configs) にまとめています
（`Configs.yaml` から自動生成）。
